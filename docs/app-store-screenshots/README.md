# App Store Screenshot Templates

These templates target the macOS App Store screenshot size `2880 x 1800`.

Files:
- `template.svg`: reusable base template with neutral placeholder copy
- `screen-01-simple-one-way-sync.svg`
- `screen-02-repeatable-tasks.svg`
- `screen-03-run-sync-and-dry-run.svg`
- `screen-04-external-drives.svg`
- `screen-05-recurring-schedules.svg`
- `screen-06-advanced-control.svg`

Recommended workflow:
1. Capture the app window at a clean, consistent size.
2. Open one of the SVG files in Figma, Sketch, Keynote, Affinity Designer, or Illustrator.
3. Replace the inner placeholder panel with the real app screenshot.
4. Keep the overall canvas at `2880 x 1800`.
5. Export to PNG at `1x` from the SVG canvas.

Layout notes:
- Top headline area is already positioned.
- The screenshot frame is centered and sized for a large desktop window.
- Keep the real screenshot fully inside the rounded dark frame.
- Do not redraw the app UI with AI; only replace the placeholder with a real screenshot.

Suggested export set:
- First 5 screens are enough for launch.
- Use screen 6 only if the advanced workflow feels strong enough for store visitors.
